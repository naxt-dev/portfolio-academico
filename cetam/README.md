# 🛠️ Formação Técnica - CETAM

Repositório dedicado às atividades do curso técnico no **Centro de Educação Tecnológica do Amazonas**. Aqui o foco é **mão na massa** e desenvolvimento de scripts funcionais.

### 💻 O que você encontrará aqui:
*   **Scripts de Programação:** Exercícios práticos focados em lógica de programação (C, PHP, Python).
*   **Banco de Dados:** Scripts SQL, automação de triggers e gerenciamento de dados relacionais.
*   **Back-end Prático:** Pequenas aplicações e CRUDs desenvolvidos durante os módulos técnicos.
*   **Laboratórios:** Testes de ambiente e configurações voltadas ao mercado de trabalho.

> 🚀 **Objetivo:** Foco em produtividade e resolução de problemas técnicos diretos.
