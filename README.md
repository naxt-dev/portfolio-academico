# Formação em Engenharia de Software & Programação

Este repositório centraliza todas as atividades, projetos e exercícios práticos desenvolvidos durante a minha trajetória académica na **FUCAPI** (Engenharia de Software) e no **CETAM** (Técnico em Informática com ênfase em Programação).

## 🚀 Objetivo
O foco principal deste portfólio é documentar a minha evolução técnica, consolidando conceitos de lógica de programação, estruturas de dados, Programação Orientada a Objetos (POO) e desenvolvimento backend utilizando tecnologias como Python, Java, C e SQL.

## 📂 Estrutura do Repositório
A organização segue a estrutura de instituições e períodos letivos para facilitar a navegação:

| Pasta | Conteúdo |
| :--- | :--- |
| `fucapi/` | Projetos e atividades da graduação em Engenharia de Software. |
| `cetam/` | Exercícios do curso técnico focados em programação e backend. |
| `projetos-pessoais/` | Sistemas e automações desenvolvidos fora do ambiente académico. |

## 🛠️ Tecnologias e Conceitos Explorados
* **Linguagens:** Python (3.10+), C, SQL.
* **Paradigmas:** Programação Imperativa e Orientada a Objetos (POO).
* **Banco de Dados:** MySQL.
